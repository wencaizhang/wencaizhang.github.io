> 使用 docker 快速安装 wordpress


## 卖瓜

这里卖瓜自夸一下我自己用的 [docker-wordpress](https://github.com/wencaizhang/docker-wordpress)

使用方法很简单：

```bash
git clone https://github.com/wencaizhang/docker-wordpress

cd docker-wordpress

docker-compose up -d
```
