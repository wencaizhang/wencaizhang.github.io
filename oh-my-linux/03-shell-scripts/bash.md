# 命令行

## 切换 shell

1、查看存在的shell

```bash
cat /etc/shells 
```
 

2、查看使用的shell

```bash
echo $SHELL
```
 

3、切换shell

切换bash 

```bash
chsh -s /bin/bash
```

切换zsh 
```bash
chsh -s /bin/zsh
```
