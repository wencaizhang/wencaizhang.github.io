# Linux配置全局环境变量的方法

## 1. 修改 `/etc/profile`

打开文件：
```sh
vim /etc/profile
```

在文件末尾添加如下代码：

```sh
export PATH="变量路径"
```

## 2. 修改 `.bashrc`

打开文件：
```sh
vim /root/.bashrc
```

在文件末尾添加如下代码：

```sh
export PATH="变量路径"
```
 

## 3. 直接在 `shell` 下用 `export` 命令修改

```sh
export PATH="$var_PATH"
```

查看当前系统下的所有环境变量：

```sh
export
```
