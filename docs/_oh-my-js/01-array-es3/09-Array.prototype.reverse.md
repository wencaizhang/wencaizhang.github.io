## 描述

`reverse()` 方法用于颠倒数组中元素的顺序。（修改原数组）

## 参数

无

## 返回值

返回颠倒顺序后的数组。

## 示例

```js
var numbers = [1, 2, 3, 4];
numbers.reverse();
console.log(numbers); // [4, 3, 2, 1]
```
