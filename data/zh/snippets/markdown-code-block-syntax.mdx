## 基本语法

使用三个反引号创建代码块

````
```language
your code goes here
```
````

例如你在 markdown 中这样写

````
```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Hello world</title>
  </head>
  <body>
    <h1>Hello world</h1>
  </body>
</html>
```
````

渲染之后会变成这样

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Hello world</title>
  </head>
  <body>
    <h1>Hello world</h1>
  </body>
</html>
```

## 显示文件名


在 `language` 后面增加 `:filename.ext` 就会渲染出文件名，两个字段中间没有空格

例如你在 markdown 中这样写

````
```html:index.html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Hello world</title>
  </head>
  <body>
    <h1>Hello world</h1>
  </body>

```
````

渲染之后会变成这样


## 行高亮


在 `language` 字段后面增加 `{numbers}` 字段，两个字段之间要有空格

> 除了单行以外，你也可指定多行，行数区间，或是两者都指定。
>
> - 行数区间: 例如 `{5-8}`, `{3-10}`, `{10-17}`
> - 多个单行: 例如 `{4,7,9}`
> - 行数区间与多个单行: 例如 `{4,7-13,16,23-27,40}`

例如你在 markdown 中这样写

````
```html {1,4-6,10}
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Hello world</title>
  </head>
  <body>
    <h1>Hello world</h1>
  </body>
</html>
```
````

渲染之后会变成这样

```html {1,4-6,10}
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Hello world</title>
  </head>
  <body>
    <h1>Hello world</h1>
  </body>
</html>
```

## 显示行号

在 `language` 字段后面增加 `showLineNumbers` 字段，两个字段之间要有空格。

例如你在 markdown 中这样写

````
```html showLineNumbers

<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Hello world</title>
  </head>
  <body>
    <h1>Hello world</h1>
  </body>
</html>
```
````

